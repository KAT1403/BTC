# BTC Markets Landing Page

A modern, pixel-perfect cryptocurrency exchange landing page. This project focuses on high-performance typography, clean UI, and a fully adaptive layout for all device types.

## Features
* **Full Responsiveness:** Adaptive design from 4K monitors to mobile devices.
* **Modern Layout:** Built using CSS Grid and Flexbox for precise alignment.
* **Optimized UI:** High-fidelity crypto price table and feature blocks.
* **Clean Code:** Semantic HTML5 and modular CSS3.
* **Typography:** Integrated with Google Fonts (Lexend).

## Technologies
* **HTML5:** Semantic markup for SEO and accessibility.
* **CSS3:** Custom properties, advanced selectors, and Media Queries.
* **Google Fonts:** Lexend font family integration.
